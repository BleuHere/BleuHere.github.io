CSS: MARGINS, BORDERS AND PADDING

by Bleuette

CSS uses the box model to organise spaces in and around an HTML element. Understanding how the margins, borders and padding of an element functions is really useful when designing a webpage.

I've seen a lot of box model images, but the squares and lines get me all confused! So I thought I'd explain the box model using something a little different...

Here we have a very handsome horse standing on some grass. His name is Larry and I've finally found him after searching for hours! Larry jumped the fence and has been AWOL for ages and must be missing home. Let's build a fence around Larry.

BORDER

Nice! Now Larry can't wander off while we make sure he's okay. The fence is a CSS border. By default, a border puts a 3px solid black line around all four sides of a selected element (Larry on his grass is all one image element).

Let's make Larry's fence brown and a little thicker.

We can change the border thickness, the style and the colour of the border like this:

border: _px style colour;

There are a bunch of border styles like dotted and double, but Larry likes to keep it simple so we'll stick with solid. Here's more info about borders. The main thing to remember is that borders separate the inside of an element from what is outside the element.

Another good analogy is that the border is like the skin. It is the divide between what is inside of you like your guts and bones, and what is outside like air, and nail polish.

PADDING

Honestly, Larry can barely turn around in this tiny paddock, so let's increase the size so that he can graze about. We are going to increase the padding. Padding is a CSS style that creates space around an object inside it's border.

We can add padding to each side for example - padding-left: 50px; or we can add it to all sides using jsut one px measurement like this: padding: 100px;.

If the measurements for the top and bottom are the same, and the measurements for the left and right are the same, we can right it like this: padding: 50px 20px;. The first number is the top and bottom measurement, and the second number is the left and right measurements.
